# BudgetFlow

BudgetFlow is a full-stack personal finance platform designed to help users track spending, manage monthly budgets, and understand financial patterns through analytics.

## Why I Built This

This project was built as a software engineering portfolio project to demonstrate backend API design, relational database modeling, authentication, testing, CI/CD, and full-stack development.

## Features

- User registration and JWT authentication
- Transaction CRUD
- Category management
- Monthly budget tracking
- Spending analytics
- PostgreSQL database
- RESTful API design
- Automated backend testing
- CI/CD with GitHub Actions

## Tech Stack

- Backend: Java Spring Boot
- Frontend: React + TypeScript
- Database: PostgreSQL
- Testing: JUnit, Mockito
- CI/CD: GitHub Actions

## System Architecture

Frontend → REST API → Service Layer → Repository Layer → PostgreSQL
